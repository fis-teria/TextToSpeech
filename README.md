C++のGUIフレームワークであるQtを使用し、機械学習部にはBoostを使ってPythonコードを埋め込んでいる。

## プロジェクトについて
このプロジェクトは誰でも機械学習ができ、誰でも手軽に扱うことができるTextToSpeechソフトウェアである。
